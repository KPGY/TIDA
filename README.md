해당 어플은 nextron 기반으로 만든 데스크톱 타임라인 기능입니다.

본인이 그때그때 생각나는 내용을 작성해 언제 작성했는지 확인이 가능한게 주 목적입니다.

해당 페이지는 투두리스트를 작성해서 관리하는 페이지입니다.

투두리스트 작성(서브 리스트 작성)/ 서브 추가 / 수정 / 삭제 / 완료 / 순서변경이 가능합니다.
